
📏 Arduino Ultrasonic Distance Sensor with LED
This project measures distance using an HC-SR04 ultrasonic sensor and displays a visual warning using an LED when an object is too close.
---
✨ Features
📐 Real-time distance measurement (in cm)
⚡ Fast and accurate readings
🚨 LED warning system for close objects
🧩 Simple and beginner-friendly design
---
🛠️ Components
Arduino Uno
HC-SR04 Ultrasonic Sensor
LED
330Ω Resistor
Jumper wires
Breadboard
---
🧠 How It Works
The ultrasonic sensor sends a sound wave and measures the time it takes to return. Using this time, the distance is calculated.
If the measured distance is less than 10 cm, the LED turns on as a warning indicator.
---
📊 Example Output

Distance: 12 cm
Distance: 8 cm → 🚨 LED ON
